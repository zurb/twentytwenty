twentytwenty

The easiest way to highlight the differences between 2 images

twentytwenty works with AngularJS and jQuery. To use the plugin just include the plugin like so:

AngularJS

<script src='angular.min.js'></script>
// Add twentytwenty after AngularJS
<script src='angular-twentytwenty.min.js'></script>

Then in your markup just add:

<twentytwenty
  before='path/to/before.jpg'
  after='path/to/after.jpg'>
</twentytwenty>

jQuery

<script src='jquery.min.js'></script>
// Add twentytwenty after jQuery
<script src='jquery.twentytwenty.min.js'></script>

Then in your markup just add:

<div class="twentytwenty-container">
  <img src="img/1_1.jpg" />
  <img src="img/1_2.jpg" />
</div>

And initialize like so:

<script>
  $(function(){
    $(".twentytwenty-container").twentytwenty();
  });
</script>
